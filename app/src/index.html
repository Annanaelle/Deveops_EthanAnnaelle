Complete API documentation has been created using Swagger

CRUD Operations

POST /user

Use this method to add a new user to the REDIS DB. In the post request, send a JSON body containing the following information: {"username" = "", "firstname" = "", "lastname"=""}

PUT /user/updates

Update the information of an existing user. Send the following JSON body: {"username" = "", "firstname" = "", "lastname"=""}

GET /user/:username

Retrieve a specific user by username from the REDIS DB.

DELETE /user/:username

Delete a specific user by username from the REDIS DB.

GET /user/keys

Retrieve all keys present in the REDIS DB.

Other Routes

GET /api-docs

Obtain complete documentation of the API.

GET /health

Check the health of the application.

Important

This application requires a REDIS DB to work perfectly. If you're using this application on AZURE or in a single Docker container not linked with a REDIS container, API methods won't work for you.
